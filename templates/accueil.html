Bienvenue sur la page d'accueil de notre Jeu de briques !
Le but du jeu est simple, il suffit de remplir toutes les cases cibles de votre grille à l’aide des briques.
À chaque tour, vous pouvez choisir une brique parmi les 4 proposées dans la pioche.
Vous pouvez soit :

  • Placer une brique sur la grille si elle correspond aux cases cibles encore libres.
  • Défausser une brique.

Le jeu se termine lorsque toutes les cases cibles sont recouvertes.
Essayez de terminer en un minimum de tour pour obtenir le meilleur score.
Bonne chance !
